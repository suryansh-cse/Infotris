Big O notation (intro)

Big O describes how runtime or space grows as input size n grows — worst-case unless noted.

  • O(1) — constant: index into list
  • O(n) — linear: loop through all items
  • O(n²) — quadratic: nested loops over same data
  • O(log n) — logarithmic: binary search
# O(n)
def total(nums):
    s = 0
    for x in nums:
        s += x
    return s
