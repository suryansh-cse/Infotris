Why data structures matter

Data structures are ways to organize data. Algorithms are steps to solve problems. Together they affect speed, memory, and whether your program scales.

Real examples

  • Search in a sorted list → binary search beats checking every item
  • Counting votes → a hash map (dict) gives fast lookups
  • Undo in an editor → a stack of states

You already used lists and dicts in Python — DSA names the ideas behind those choices.
