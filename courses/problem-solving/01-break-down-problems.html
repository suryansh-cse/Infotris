Breaking down problems

~30 min

Start with the outcome

Before typing code, state in one sentence what success looks like. Example: "Print the largest number in a list."

Split into sub-problems

  1. Get input (list of numbers)
  2. Track the largest so far
  3. Loop and compare
  4. Print result

Edge cases

  • Empty list?
  • One item?
  • Negative numbers?

Practice

Break down: "Count how many words in a sentence start with vowels." Write steps in plain English only.
