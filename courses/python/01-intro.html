Lesson 1 · Python

Introduction to Python

~25 min

What is Python?

Python is a readable, versatile language used for web apps, automation, data science, scripting, and AI tools. Its syntax is closer to plain English than many languages, which makes it a strong first language.

How to run Python

  • Install Python from python.org (3.10+ recommended).
  • Open a terminal and type python or python3 for an interactive shell.
  • Or save code in a file ending in .py and run: python myfile.py

Your first program

print("Hello, OpenLearn!")

print() sends text to the screen. Strings go inside quotes.

Comments

# This line is ignored — use comments for notes
print("Code runs")  # comment after code works too

Practice
