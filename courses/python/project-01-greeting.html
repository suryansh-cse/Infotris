Project 1 · Week 1

Personal greeting script

~45 min · Learn + build + verify

Milestone 1 Goal

Build a script that functions as a Personal Profile & Study Calculator. The script prompts the user for their name, birth year, hourly rate, and weekly study hours. It then calculates their age, estimates yearly learning hours, and displays a formatted CLI profile card using f-strings.

Requirements

  • Ask for name, birth year, hourly rate, and weekly study hours using input().
  • Convert numerical inputs from string to int and float.
  • Calculate age (using the current year 2026) and annual study hours (weekly hours * 52).
  • Print a visual, multi-line card banner with user details using f-strings or triple quotes.

Starter Template

name = input("What is your name? ")
birth_year = int(input("What is your birth year? "))
weekly_hours = float(input("How many hours do you study per week? "))

age = 2026 - birth_year
yearly_hours = weekly_hours * 52

print(f"""
----------------------------------------
       DEVELOPER PROFILE CARD
----------------------------------------
Name:         {name}
Age:          {age} years old
Weekly Study: {weekly_hours} hours
Yearly Study: {yearly_hours} hours
----------------------------------------
""")
