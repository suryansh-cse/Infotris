Project 1 · Week 1

Personal greeting script

~45 min

Goal

Build a script that asks for the user's name and goal, then prints a personalized motivational message.

Requirements

  • Use input() at least twice
  • Store answers in variables
  • Print a multi-line message with f-strings

Starter template

name = input("What is your name? ")
goal = input("What do you want to learn? ")

print(f"""
Hi {name}!
Welcome to OpenLearn.
Let's work toward: {goal}
You've got this.
""")

Stretch goals

  • Ask for hours per week and estimate weeks to finish a 40-hour path
  • Save the message to greeting.txt
