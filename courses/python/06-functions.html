Lesson 6

Functions and return values

~40 min

Defining functions

def greet(name):
    return f"Hello, {name}!"

message = greet("Alex")
print(message)

Parameters and arguments

def add(a, b):
    return a + b

print(add(3, 5))

Default parameters

def power(base, exponent=2):
    return base ** exponent

print(power(4))      # 16
print(power(2, 5))   # 32

Why functions matter

They reduce duplication, clarify intent, and make testing easier — essential for larger programs and projects.

Practice

Write celsius_to_fahrenheit(c) and is_even(n). Test with several inputs.
