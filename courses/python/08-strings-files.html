Lesson 8

Strings and reading files

~40 min

String methods

text = "  OpenLearn  "
print(text.strip().lower())
print("python".replace("p", "P"))

f-strings

topic = "Python"
hours = 5
print(f"You studied {topic} for {hours} hours.")

Reading a file

with open("notes.txt", "r", encoding="utf-8") as file:
    content = file.read()
    print(content)

Writing a file

with open("log.txt", "w", encoding="utf-8") as file:
    file.write("Session complete\n")

with closes the file automatically — always prefer it.

Practice

Write a program that counts lines in a text file you create manually.
