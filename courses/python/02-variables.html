Variables and data types

~30 min

Variables

A variable is a name that points to a value. You assign with =.

name = "Sam"
age = 17
height = 5.9
is_student = True

Common types

  • str — text in quotes
  • int — whole numbers
  • float — decimals
  • bool — True or False

Checking types

print(type(name))   # <class 'str'>
print(type(age))    # <class 'int'>

Input from the user

city = input("Where do you live? ")
print("Nice to meet someone from", city)

input() always returns a string. Convert numbers with int() or float().

Ask for name and birth year, compute approximate age, print a one-line introduction.
