Lesson 7

Dictionaries and key-value data

~35 min

What is a dictionary?

A dict maps unique keys to values — perfect for records, counts, and lookups.

user = {
    "name": "Riya",
    "email": "riya@example.com",
    "courses": 3
}

print(user["name"])
user["courses"] = 4

Looping over dicts

for key, value in user.items():
    print(key, "→", value)

Safe access

phone = user.get("phone", "not set")

Practice

Create a dict for one contact (name, phone, city). Print a formatted card using the values.
