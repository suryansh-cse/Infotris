Lesson 5

Lists and list methods

~35 min

Creating lists

tasks = ["study", "practice", "review"]
numbers = [10, 20, 30]
mixed = ["Py", 3, True]

Indexing and slicing

print(tasks[0])      # first item
print(tasks[-1])     # last item
print(tasks[1:3])    # slice — items at index 1 and 2

Useful methods

  • append(x) — add to end
  • remove(x) — remove first match
  • pop() — remove and return last
  • len(list) — number of items

List comprehensions (intro)

squares = [n * n for n in range(6)]
print(squares)

Practice

Build a to-do list: start empty, append three tasks, print each with a number.
